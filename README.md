# 📊 Customer Churn Analysis Dashboard — Excel

## 📌 Project Overview
An interactive Excel dashboard analyzing customer churn patterns 
for a telecom company. Identifies key reasons why customers leave 
and provides actionable business recommendations.


## 🎯 Problem Statement
A telecom company is losing customers every month.
The goal is to:
- Find **who** is churning
- Find **why** they are churning
- Suggest **how** to reduce churn


## 📂 Dataset
- **Source:** Simulated telecom customer data
- **Size:** 10 customers, 12 columns
- **Fields:**
  - Customer ID, Gender, Age
  - Tenure (months)
  - Monthly Charges, Total Charges
  - Contract Type (Monthly/Yearly/Two-Year)
  - Payment Method
  - Internet Service (Fiber Optic/DSL)
  - Support Calls
  - Satisfaction Score
  - Churn (Yes/No)

---

## 🛠️ Tools Used
- Microsoft Excel / WPS Office
- Formulas: COUNTIF, COUNTIFS, AVERAGEIF, COUNTA
- Charts: Pie Chart, Column Chart
- Conditional Formatting
- KPI Dashboard Design

---

## 📊 Dashboard Features
✅ KPI Summary Boxes
- Total Customers
- Total Churned
- Churn Rate %
- Retention Rate

✅ Pie Chart — Churned vs Retained

✅ Column Chart — Churn by Contract Type

✅ Column Chart — Churn by Internet Service

✅ Conditional Formatting
- 🔴 Red rows = Churned customers
- 🟢 Green rows = Retained customers

✅ Auto-updating date using =TODAY()

---

## 🔍 Key Findings

| Finding | Insight |
|---|---|
| Churn Rate | 50% overall |
| Monthly Contract | 5/5 churned — highest risk! |
| Yearly Contract | 0/3 churned — most loyal! |
| Fiber Optic | 4x more churn than DSL |
| Avg Support Calls | Churned = 5.4 vs Retained = 1.6 |
| Avg Satisfaction | Churned = 4.6 vs Retained = 8.4 |

---

## 💡 Business Recommendations

1. **Convert monthly customers to yearly plans**
   → Offer discounts for long-term commitment

2. **Improve Fiber Optic service quality**
   → High churn suggests service dissatisfaction

3. **Reduce support call wait times**
   → Churned customers averaged 5+ support calls

4. **Target low satisfaction customers proactively**
   → Satisfaction score below 6 = high churn risk
